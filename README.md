# @steijnveer/create-fbr-plugin

Create a new file-based-router plugin with one command.

## Usage

```bash
# npm
npm create @steijnveer/fbr-plugin my-plugin

# yarn
yarn create @steijnveer/fbr-plugin my-plugin

# pnpm
pnpm create @steijnveer/fbr-plugin my-plugin

# bun
bun create @steijnveer/fbr-plugin my-plugin
```

Or create in the current directory:

```bash
npm create @steijnveer/fbr-plugin .
```
